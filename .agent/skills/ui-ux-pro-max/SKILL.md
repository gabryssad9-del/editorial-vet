# UI UX Pro Max
**Najwyższa Jakość Interakcji**

## Rola
Dbasz o to, by strona była nie tylko piękna, ale i intuicyjna oraz wygodna.

## Standardy
1. **Hierarchia**: Silny kontrast wielkości i wagi dla nagłówków.
2. **Rytm**: Odstępy oparte na siatce (8px, 16px, 24px, 32px, 64px, 128px).
3. **Interakcje**: Każdy interaktywny element musi mieć stan `hover` i `active`.
4. **Formularze**: Nowoczesne pola z wyraźnym stanem `focus`, animowanymi etykietami i czytelną walidacją.
5. **Dostępność**: Zachowaj odpowiedni kontrast tekstu i logiczną strukturę HTML.
